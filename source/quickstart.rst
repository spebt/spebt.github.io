===========
Quick Start
===========

.. important::
   :class: sd-fs-6

   | :octicon:`info;1.5em;sd-text-info` `Slack <https://slack.com/>`_ is the in-group realtime communication tool. 
   | *Contact the* `project administrator <pages/contributors.html#project-administrators>`_ *for an invitation*

   | :octicon:`info;1.5em;sd-text-info` For document file storage and sharing, `UB Box <https://buffalo.box.com>`_ is used. 
   | *Contact the* `project administrator <pages/contributors.html#project-administrators>`_ *for project folder access*

   | :octicon:`info;1.5em;sd-text-info` The `High Performance Computer (HPC) compute cluster <https://docs.ccr.buffalo.edu/en/latest/hpc/clusters/#ub-hpc-compute-cluster>`_ at the State University of New York (SUNY) at Buffalo is used for the analytical modelling and Monte Carlo simulation.
   | Please take the `Intro to CCR course <https://ublearns.buffalo.edu/d2l/le/discovery/view/course/209035>`_ first.
